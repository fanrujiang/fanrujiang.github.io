    Spring

    Spring 是一个开源的轻量级应用开发框架，提供全面的基础设施支持和丰富的特性，简化开发并提高代码的可测试性和可维护性。

    AJAX

    Ajax是前端技术，可在不刷新页面的情况下与服务器交换数据，提高用户体验。

    JavaSE

    JavaSE是Java平台标准版，提供核心功能和基础库，用于开发跨平台的桌面和嵌入式应用程序。

    Linux

    Linux是一种开源操作系统，以稳定、安全和灵活著称，广泛用于服务器和嵌入式系统。

    Docker

    Docker是一种容器化平台，用于打包、发布和运行应用程序，提供高效的跨平台部署解决方案。

    Nginx

    Nginx 是开源、高性能、高可靠的 Web 和反向代理服务器。

    nodeJS

    Node.js是一个基于Chrome V8引擎的JavaScript运行时环境，用于构建高性能、可扩展的网络应用程序。

    Elasticsearch

    Elasticsearch是一个开源的分布式搜索和分析引擎，用于快速搜索、分析和可视化大规模数据。

    RabbitMQ

    RabbitMQ是一款使用Erlang语言开发的的开源消息中间件，，用于在分布式系统中存储转发消息。
